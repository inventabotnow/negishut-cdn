# Negishut CDN

Distribution repository for the Negishut accessibility widget (רכיב נגישות).
This repo contains the built file only.

## Usage

```html
<script src="https://cdn.jsdelivr.net/gh/inventabotnow/negishut-cdn@1/negishut.js" defer
        data-site-id="YOUR-SITE-ID"
        data-org="שם העסק"
        data-coordinator-name="שם רכז הנגישות"
        data-coordinator-email="a11y@example.co.il"
        data-coordinator-phone="03-1234567"></script>
```
